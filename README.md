🚀 Overview

Kisan Connect is a mobile-first Progressive Web App (PWA) that combines Machine Learning, Generative AI, and real-world agricultural intelligence to guide farmers toward smarter and more profitable decisions.

Instead of just giving raw data, it delivers:

clear advice
local language support
context-aware recommendations

👉 Think of it as a digital farming assistant that evolves with the farm itself.

🎯 Problem We Solve

Farmers today often face:

fragmented and hard-to-understand data
lack of localized and real-time guidance
no clear insight into profitability vs crop choice

👉 Kisan Connect bridges this gap by combining AI + context + accessibility into one unified platform.

# Mac/Linux

uvicorn app:app --reload --port 3000
2️⃣ Frontend

npm install
npm run dev
